This repository contains a Python script designed to track the monthly removal of pine trees (invasive exotic species) from Rio Vermelho State Park, located in Florianópolis, Brazil.  
The goal is to provide data and insights that contribute to ecological restoration and the preservation of native biodiversity.

### Features
- Records monthly removal data  
- Generates visualizations  
- Serves as a tool for environmental monitoring and research
